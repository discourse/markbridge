## Install

```bash
bundle add markbridge
```

## See it in action

```ruby
require "markbridge/bbcode"
require "markbridge/html"

Markbridge.bbcode_to_markdown("[b]Hello[/b] [url=https://example.com]world[/url]!")
# => "**Hello** [world](https://example.com)!"

Markbridge.html_to_markdown("<p>Nested <em>is</em> fine.</p>")
# => "Nested *is* fine."
```
